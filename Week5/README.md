# การวิเคราะห์ความเหมาะสมของพื้นที่เพื่อจัดตั้งสถานีวิจัยจำลองดาวอังคารในภาคตะวันออกเฉียงเหนือ
## บทนำ
การค้นหาพื้นที่จำลองดาวอังคาร (Mars Analog Site) เป็นแนวทางที่ใช้ในการศึกษาพื้นที่บนโลกที่มีลักษณะคล้ายคลึงกับพื้นผิวดาวอังคาร เพื่อใช้เป็นพื้นที่ทดสอบและศึกษาด้านรีโมตเซนซิงและภูมิสารสนเทศ ภาคตะวันออกเฉียงเหนือของประเทศไทยมีลักษณะภูมิประเทศแบบที่ราบสูง ดินสีแดงจากเหล็กออกไซด์ และมีช่วงฤดูแล้งที่ยาว ซึ่งมีศักยภาพในการใช้เป็นพื้นที่จำลองดาวอังคาร งานวิจัยนี้จึงมุ่งวิเคราะห์ความเหมาะสมของพื้นที่ในภาคตะวันออกเฉียงเหนือโดยใช้ข้อมูลจาก Google Earth Engine เพื่อระบุพื้นที่ที่มีลักษณะใกล้เคียงกับพื้นผิวดาวอังคารมากที่สุด

## 1. Problem Statement
โครงการนี้มีวัตถุประสงค์เพื่อวิเคราะห์ความเหมาะสมของพื้นที่จำลองดาวอังคารในภาคตะวันออกเฉียงเหนือของประเทศไทย ซึ่งมีลักษณะภูมิประเทศแบบที่ราบสูงและดินสีแดงที่อาจมีความคล้ายคลึงกับพื้นผิวดาวอังคาร การศึกษานี้ใช้ข้อมูลรีโมตเซนซิงจาก Google Earth Engine โดยคำนวณดัชนี Iron Oxide Index เพื่อระบุพื้นที่ดินสีแดง ใช้ NDVI เพื่อกรองพื้นที่ที่มีพืชพรรณหนาแน่น และใช้ค่าความลาดชันจากข้อมูล DEM เพื่อเลือกพื้นที่ที่มีลักษณะราบเรียบ การวิเคราะห์ดังกล่าวมีเป้าหมายเพื่อคัดเลือกพื้นที่ที่มีศักยภาพสำหรับใช้เป็นพื้นที่จำลองดาวอังคาร ซึ่งสามารถสนับสนุนการศึกษาด้านรีโมตเซนซิง การวิเคราะห์เชิงพื้นที่ และการทดลองภาคสนามในบริบทประเทศไทย ทั้งนี้ การศึกษาใช้ข้อมูลที่มีอยู่ใน Google Earth Engine ทำให้สามารถดำเนินการได้ภายในระยะเวลาที่กำหนดโดยไม่ต้องเก็บข้อมูลภาคสนามเพิ่มเติม


## 2. Research Questions
RQ1: พื้นที่ใดในภาคตะวันออกเฉียงเหนือที่มีค่าดัชนีเหล็กออกไซด์สูงและพืชพรรณต่ำที่สุด
RQ2: พื้นที่ที่มีความเหมาะสมสูงสุดมีขนาดพื้นที่เท่าใด
RQ3: พื้นที่ที่มีความเหมาะสมสูงสุดกระจุกตัวอยู่บริเวณจังหวัดใด
เหตุผล
คำถามวิจัยถูกออกแบบให้ตอบได้ด้วยข้อมูลเชิงพื้นที่และการวิเคราะห์จากดัชนีรีโมตเซนซิง โดย RQ1 ใช้ระบุลักษณะพื้นผิว RQ2 ใช้ประเมินขนาดพื้นที่ และ RQ3 ใช้ระบุการกระจายตัวเชิงพื้นที่

## 3. Data Inventory
ภาพดาวเทียมที่ใช้คือ Landsat 8 Collection 2 Level 2
ช่วงเวลา: 1 มีนาคม 2023 – 30 เมษายน 2025
เหตุผลที่เลือกช่วงเวลา
เลือกช่วงฤดูแล้ง เนื่องจากพืชพรรณมีความหนาแน่นต่ำ ทำให้สามารถเห็นพื้นผิวดินได้ชัดเจน และลดความสับสนระหว่างพื้นที่เกษตรกับพื้นที่ดินเปล่า นอกจากนี้การใช้ข้อมูลหลายปีช่วยลดผลกระทบจากเมฆและเพิ่มความเสถียรของผลลัพธ์
เหตุผลที่ไม่เลือกช่วงฤดูเกษตร
ช่วงที่พืชเติบโตเต็มที่จะมีค่า NDVI สูง ทำให้พื้นผิวดินถูกปกคลุม และไม่สามารถแยกพื้นที่ดินจริงได้อย่างชัดเจน จึงเลือกช่วงฤดูแล้งแทน
Dataset ที่ใช้
Landsat 8 → คำนวณ Iron Oxide, NDVI, BSI


NASADEM → คำนวณ Slope


ESA WorldCover → Mask ป่า เมือง น้ำ


เหตุผลการเลือก Dataset
เลือก Landsat 8 เนื่องจากมี band ที่เหมาะสำหรับคำนวณดัชนีดินและพืชพรรณ
เลือก NASADEM เนื่องจากมีความละเอียด 30 เมตร เหมาะกับการคำนวณ slope
เลือก ESA WorldCover เพื่อใช้ตัดพื้นที่ที่ไม่ใช่พื้นผิวดินออกจากการวิเคราะห์
ดัชนีที่คำนวณ
Iron Oxide Index = Red / Blue
เหตุผล: ใช้ตรวจสอบดินที่มีเหล็กออกไซด์สูงซึ่งคล้ายพื้นผิวดาวอังคาร
NDVI = (NIR - Red) / (NIR + Red)
เหตุผล: ใช้กรองพื้นที่ที่มีพืชพรรณหนาแน่นออก
BSI = ((SWIR + Red) - (NIR + Blue)) / ((SWIR + Red) + (NIR + Blue))
เหตุผล: ใช้ยืนยันพื้นที่ดินเปล่า

## 4. Methodology Flow
1.กำหนดพื้นที่ภาคตะวันออกเฉียงเหนือ เพื่อจำกัดขอบเขตการวิเคราะห์


2.ดึงข้อมูล Landsat 8 ช่วงฤดูแล้ง เพื่อลดพืชพรรณ


3.ทำ cloud mask เพื่อลดผลกระทบจากเมฆ


4.คำนวณ Iron Oxide เพื่อหาดินสีแดง


5.คำนวณ NDVI เพื่อกรองพืชพรรณ


6.คำนวณ BSI เพื่อยืนยันดินเปล่า


7.คำนวณ slope เพื่อเลือกพื้นที่ราบ


8.mask ป่า เมือง น้ำ เพื่อเหลือเฉพาะพื้นผิวดิน


9.normalize ค่า เพื่อให้ตัวแปรอยู่ในช่วงเดียวกัน


10.weighted overlay เพื่อรวมตัวแปร


11.สร้าง suitability map เพื่อแสดงพื้นที่เหมาะสม



Weighted Overlay
Iron oxide = 40%
Slope = 40%
NDVI = 20%
เหตุผลการกำหนดน้ำหนัก
ใช้วิธี expert-based weighting โดยให้ความสำคัญกับลักษณะพื้นผิวดินและความราบเรียบเป็นหลัก เนื่องจากเป็นคุณสมบัติสำคัญของพื้นที่จำลองดาวอังคาร ขณะที่ NDVI ใช้เป็นตัวกรองพืชพรรณจึงให้ค่าน้ำหนักน้อยกว่า
